# Polestar Static Microsite

An independent editorial microsite dedicated to Polestar, built with static HTML, CSS, and a small vanilla JavaScript interaction.

## Coding Style

- Semantic HTML for structure.
- Modern responsive CSS with no framework or build step.
- Vanilla JavaScript for the drive brief tab interaction.
- Local official press images stored in `assets/images/`.

## Free Hosting Pick

Use Netlify for this version. Drag this folder into Netlify Drop or connect it to a GitHub repo. Because there is no build step, the publish directory is the project root and the entry file is `index.html`.
